CommandProjectUniversal
Описание проекта
CommandProjectUniversal - это приложение для управления интернет-услугами, разработанное в рамках командного проекта по управлению программными проектами. 
Приложение предоставляет клиентский интерфейс для работы с базой данных, включая редактирование основных и вспомогательных таблиц (справочников), 
поиск, фильтрацию и составление отчётов.

Основные возможности:
Управление справочниками: Страны, провайдеры, планы услуг, клиенты, продажи.
Фильтрация услуг: По цене (от/до).
Отчеты: График суммы продаж по провайдерам (круговая диаграмма).
CRUD-операции: Добавление, редактирование, удаление записей в справочниках и услугах.
Проект реализован на C# с использованием .NET Framework 8, WPF для интерфейса, Entity Framework Core для ORM и SQLite для базы данных. 
Для визуализации графиков используется библиотека OxyPlot.

Технологии
Язык программирования: C#
Фреймворк: .NET Framework 8
Интерфейс: WPF
ORM: Entity Framework Core
База данных: SQLite
Графики: OxyPlot
Контроль версий: Git (репозиторий: ссылка на GitHub)
Структура базы данных
База данных состоит из следующих таблиц (сущностей):

Countries (Страны): Id, Name, Code
Providers (Провайдеры): Id, Name, CountryId (FK), Address, Phone, Email
ServicePlans (Планы услуг): Id, ProviderId (FK), CountryId (FK), Name, SpeedMbps, PricePerMonth, DataLimitGB, LaunchYear
Customers (Клиенты): Id, Name, Email, Phone
Services (Услуги): Id, ServicePlanId (FK), Name, ContractNumber, InstallationDate, Address, IsActive
Sales (Продажи): Id, ServiceId (FK), CustomerId (FK), SaleDate, SalePrice, Description


База данных автоматически мигрируется при запуске приложения (в App.xaml.cs).
Если нужно вручную: Используйте Entity Framework Tools в Package Manager Console:
Update-Database
Запустите приложение:

Скомпилируйте и запустите проект из Visual Studio.
Главное окно откроется с загруженными данными.
Backup базы данных
База данных заполняется seed-данными через Entity Framework Core в файле Data/AppDbContext.cs (метод OnModelCreating). Данные включают:

3 страны (Россия, Украина, Беларусь).
3 провайдера (Ростелеком, МТС, Билайн).
9 планов услуг.
10 клиентов.
15 услуг.
5 продаж.
Для резервного копирования: Скопируйте файл internet_services.db (создается в корне проекта).
Альтернативно, используйте SQL-скрипт из миграций (Migrations/20251121102842_ServiceManagement.cs).

Использование
Главное окно
Фильтрация: Введите минимальную и максимальную цену, нажмите "Фильтровать".
Управление услугами: Добавление, редактирование, удаление услуг.
Отчёт: Нажмите "Отчёт" для просмотра круговой диаграммы продаж по провайдерам.
Главная страница: Возврат к списку услуг.
Меню "Справочники"
Страны: Добавление/редактирование/удаление стран.
Провайдеры: Управление провайдерами (связаны со странами).
Планы услуг: Управление планами (связаны с провайдерами и странами).
Клиенты: Управление клиентами.
Продажи: Управление продажами (связаны с услугами и клиентами).
Структура проекта
Data/: AppDbContext.cs - контекст базы данных.
Migrations/: Миграции EF Core.
Models/: Классы сущностей (Country, Provider, etc.).
ViewModels/: MainViewModel.cs - модель представления.
Views/: XAML-файлы окон (MainWindow, EditCountryWindow, etc.) и их код-behind.
App.xaml/cs: Точка входа, миграции БД.
